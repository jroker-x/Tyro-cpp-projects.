# C++ Search Engine

A simple search engine built in modern C++.

Features:
- Recursive text file loading
- Inverted index
- Case-insensitive search
- Multi-word AND queries
- Word frequency based search
- Top N words of a .txt 

Concepts practiced:
- STL containers
- File I/O
- Hash maps
- Hash sets
- Move semantics
- Const correctness
